What does node do?

as a server
To run node a single time:
1) Create a file called hello.js with console.log('hello world')
2) Type node hello.js
3) You should see "hello world" in the console.

Wow, it does quite a bit, but can we make it do more?

First we need to learn about the Node.js package manager

Node Package Manager (npm) installs the packages you want to use and provides a useful interface to work with them

It finds a package online, then copies into a node_modules folder in your project folder

Then you can require the package in your code:
var express = require('express')

And now you can call any of the functions in that package

These npm modules help extend the basic functionality of node.js and speed up programming

Which modules do you need?

  • express - Handles http routing
  • mongodb - Connects to MongoDb
  • body-parser - Parses request bodies
  • superagent - Outbound http calls

You install them by typing:
npm install express --save

Which copies the files to your folder and saves the package name and version in your package.json file

Server Example

Let's jump right in and try

What's the worst that can happen?

Create a file called server.js in your node project folder

Make sure you've installed express.js
(npm install express --save)
Create a file called server.js in your node project folder

Write this code:

var express = require('express');
var app = express();

app.get('/', function (req, res) {
  res.send('Hello World!');
});

app.listen(3000, function () {
  console.log('Example app listening on port 3000!');
});

We just created our first Node app?
